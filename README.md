# Selenium Practical Questions

This project contains Selenium WebDriver practical examples implemented using Java.
It is created for learning, practice, and interview preparation purposes.

All Selenium automation scripts are written as per Maven standards inside `src/test/java`
and organized topic-wise for better understanding and maintenance.
